# 상세 → 목록 복귀 시 스크롤 위치 복원

날짜: 2026-04-07

## 기능 추가

- **상세 페이지에서 뒤로가기 시 스크롤 위치 복원**
  - 증상/배경: SSR 전환 후 상세 페이지에서 목록으로 돌아오면 첫 20마리만 보이고 스크롤이 맨 위로 초기화됨
  - 원인: Server Component 재렌더 시 PokemonGrid가 리셋되어 로드된 아이템과 스크롤 위치를 잃음
  - 수정/개선 내용:
    - `sessionStorage`에 extra 아이템 + 스크롤 Y 위치 저장
    - 카드 클릭 시 `pg:nav-to-detail` 플래그 저장 → 마운트 시 이 플래그가 있을 때만 스크롤 복원
    - 필터 변경 시에는 플래그 없음 → 스크롤 복원 안 함 (맨 위에서 시작)

## 주의사항

- sessionStorage는 탭 단위로 유지되며 새 탭/창에서는 복원되지 않음
- 필터 조합(q + generation + type)을 `gridKey`로 사용하여 서로 다른 필터의 상태가 혼용되지 않도록 격리
