# 다른 폼 클릭 시 페이지 이동 제거 — 인라인 전환으로 변경

날짜: 2026-04-07

## 버그 수정 / 기능 개선

- **다른 폼 클릭 시 페이지 이동(404) 문제 수정**
  - 증상: 메가 독침붕 등 폼 클릭 시 `/pokemon/10090` 으로 이동하여 404 발생
  - 원인: `AlternateForms`가 `router.replace`로 폼 ID 페이지로 이동, 해당 URL은 처리 불가
  - 수정: 페이지 이동 제거 → 동일 페이지에서 폼 데이터를 인라인으로 전환

- **상세 페이지 구조 개선**
  - `PokemonDetailView` 클라이언트 컴포넌트 신설: 폼 선택 상태 관리 및 표시 담당
  - `page.tsx`는 서버 컴포넌트로 유지 (초기 데이터 fetch 후 `PokemonDetailView`에 전달)
  - 폼 선택 시 `/api/pokemon/[id]` 호출 후 이미지/이름/타입/스탯/설명 즉시 갱신

- **`/api/pokemon/[id]` API 라우트 신설**
  - 상세 페이지 폼 전환용 API 엔드포인트 추가
